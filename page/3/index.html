LeetCode 162 - Find Peak Element

Posted on 2016-10-14 |

Solution

Binary search. If num[mid] > num[mid-1] && num[mid] > num[mid+1], then num[mid] is the peak element; otherwise if num[mid] < num[mid-1], find the peak element in the left part else find the element in the right part.

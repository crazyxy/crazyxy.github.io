LeetCode 398 - Random Pick Index


2016-9-16 LeetCode

Solution

概率题。每次pick时遍历数组，记录已经访问到的target的个数cnt（包括当前的元素）。如果当前访问的数等于target，则该下标有1/cnt的概率被选中，证明如下：

  • 归纳法。

    • 数组中只有一个元素，被选中的概率是1；

    • 数组中有两个元素，第一个元素被选中的概率是1，访问到第二个元素的时候，该元素被选中的概率是1/2，而第一个元素被选中的概率是1*1/2；

    • 假设当数组中有k个元素，被选中的元素的概率是1/k；

    • 当遇到第k+1个元素时，第k+1个元素被选中的概率为1/(k+1)，而之前每个元素被选中的概率都是(1-1/(k+1))*1/k=1/(k+1);

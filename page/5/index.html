LeetCode 124 - Binary Tree Maximum Path Sum

Posted on 2016-10-12 |

Solution

DFS similar. Path with maximum sum in tree rooted at u has following possibilities:

- The path crosses the `u`
- The path is in the left child
- The path is in the right child

We need to record four values:

- Maximum sum from the left child to the root `left`
- Maximum sum from the right child to the root `right`
- Maximum sum in the left child `lret`
- Maximum sum in the right child `rret`

The answer is max(left+right-root->val, lret, rret)

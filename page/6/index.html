LeetCode 101 - Symmetric Tree

Posted on 2016-10-11

Solution

To determine whether a tree is symmetric is to check whether two trees are mirrors of each other. Firstly, we check if the root contains the same value; secondly, we check whether the left child of Left and right child of Left are mirrors and the right child of Left and left child of Right are mirrors.

LeetCode 46 - Permutations

Posted on 2016-10-11

Solution

To find permutations of [1..n], we can divide the task into three parts. Firstly, we select an element k; secondly, we find the permutations of remaining elements; thirdly, combines the results with the selected element k.

LeetCode 115 - Distinct Subsequences

Posted on 2016-10-10

Solution

动态规划。dp[i][j]表示s中前i个字符中包含的t的前j个字符的子串的个数。因此有dp[i][j] = dp[i-1][j] + dp[i-1][j-1] * (s[i-1] == t[j-1])
